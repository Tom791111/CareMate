# CareMate 操作手冊

## 1. 啟動系統
執行 `controller.Main.java`，進入 CareMate 身分選擇頁。

## 2. 登入
可使用測試帳號：
- family / 1234
- elder / 1234
- caregiver / 1234
- admin / 1234

## 3. 註冊
於登入頁點選註冊，輸入帳號、密碼、姓名、電話、Email 與角色後建立帳戶。

## 4. Dashboard 2.0
首頁會依 MySQL 資料自動計算：
- 今日完成率
- 健康分數
- 血壓體溫
- 用藥狀態
- 飲食狀態
- 情緒狀態

## 5. 鈴鐺提醒
點右上角鈴鐺，可查看今日尚未紀錄項目，並點選前往補登。

## 6. CRUD 功能
健康、用藥、飲食、情緒、任務、報表等頁面皆可操作新增、修改、刪除、查詢。

## 7. 照顧安排
點選日期後，可新增行程內容、開始時間、結束時間、地點/類型與備註。

## 8. 報表列印
進入報表查詢頁，查詢資料後按「列印」。
