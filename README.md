# CareMate 長照照顧管理系統

## 📌 專案介紹

CareMate 是一套以長期照護為核心概念所設計的桌面應用程式，提供照護人員與家屬快速記錄長輩照顧資訊。

本專案使用 Java Swing 開發圖形化介面（GUI），模擬長照機構常見的照顧紀錄流程，包含：

* 個案基本資料管理
* 今日照顧任務追蹤
* 心情狀態紀錄
* 重要提醒事項
* 列印照顧紀錄
* 即時系統時間顯示

---

## 🖥️ 系統畫面

### CareMate 長照照顧首頁

功能區塊包含：

### 👤 個案資訊

* 姓名
* 年齡

### 📋 今日照顧任務

* 用餐協助
* 飲食記錄

狀態選項：

* 已完成
* 待執行

### 😊 今日心情

心情評估等級：

* 很差
* 差
* 普通
* 好
* 很好

### 🔔 重要提醒

可輸入：

* 用藥提醒
* 血壓量測提醒
* 回診提醒
* 特殊照護事項

### ⏰ 系統時間

每秒自動更新目前時間

---

## ✨ 功能特色

### 即時時間顯示

透過 Java Timer 每秒更新系統時間。

### 個案資料管理

可輸入與修改：

* 姓名
* 年齡

### 照顧任務追蹤

可記錄：

* 用餐協助狀態
* 飲食紀錄狀態

### 心情紀錄

透過下拉式選單快速紀錄長輩情緒。

### 重要提醒功能

可輸入照護備註與提醒事項。

### 列印功能

支援：

* 個案姓名
* 年齡
* 心情狀態
* 重要提醒

列印前可預覽系統列印視窗。

## 🚀 執行方式

### Eclipse

1. 匯入專案
2. 開啟 HomeworkUI.java
3. 點選 Run As → Java Application

### Command Line

```bash
javac HomeworkUI.java
java HomeworkUI
```
